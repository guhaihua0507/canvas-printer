# canvas-printer

###start
```
java -cp canvas-printer-1.0-SNAPSHOT.jar com.ghh.canvas.Main
```

### usage
```
h/help              print help
C w h 			    Create new canvas
L x1 x2 y1 y2 		Create a new line from (x1,y1) to (x2,y2). Currently only horizontal or vertical lines are supported.
R x1 y1 x2 y2 		Create a new rectangle, whose upper left corner is (x1,y1) and lower right corner is (x2,y2). 
Q 					Quit the program.
```
